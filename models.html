models¶

Query PyTorch model information.

torchutils.models.get_model_param_count(model)¶

Count total parameters in the PyTorch model.

Parameters:model (nn.Module) – PyTorch model.
Returns:Number of parameters in the model.
Return type:int

Example:

import torchvision
import torchutils as tu

model = torchvision.models.alexnet()
total_params = tu.get_model_param_count(model)
print('Total model params: {:,}'.format(total_params))

Out:

Total model params: 61,100,840
torchutils.models.get_model_flops(model, input_size, device='cpu', unit='FLOP')¶

Count total FLOPs for the PyTorch model.

Parameters:
  • model (nn.Module) – PyTorch model.
  • input_size (tuple) – Input size to the model. Can be a list of multiple inputs.
  • device (str) – Device to map the checkpoint, “cpu” or “cuda”. (default: ‘cpu’)
  • unit (str) – FLOPs unit. Can be ‘FLOP’, ‘MFLOP’ or ‘GFLOP’. (default: ‘FLOP’)
Returns:

Number of FLOPs.
Return type:

float

Example:

import torchvision
import torchutils as tu

model = torchvision.models.alexnet()
total_flops = tu.get_model_flops(model, (3, 224, 224), unit='GFLOP')
print('Total model FLOPs: {:,} GFLOPs'.format(total_flops))

Out:

Total model FLOPs: 0.77 GFLOPs
torchutils.models.get_model_summary(model, input_size, batch_size=-1, device='cpu')¶

Print model summary.

Parameters:
  • model (nn.Module) – PyTorch model.
  • input_size (tuple) – Input size to the model. Can be a list of multiple inputs.
  • batch_size (int) – Batch size. (default: -1)
  • device (str) – Device to map the checkpoint, “cpu” or “cuda”. (default: ‘cpu’)
Returns:

import torchvision
